# Troubleshooting Guide

Common issues and their solutions when using Codebuff.

## Connection Issues

If you're having trouble connecting:
1. Check your internet connection
2. Verify your authentication token
3. Try restarting Codebuff
4. Verify that Codebuff is up to date

## Permission Errors

When encountering permission errors:
1. Check file permissions
2. Verify you're in the correct directory
3. Ensure you have write access
